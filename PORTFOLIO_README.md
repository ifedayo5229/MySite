## üé® Customization

### 1. Add Your Profile Image
Place your professional photo at:
```
src/assets/images/profile.jpg
```

**Image specifications:**
- High quality (at least 800x1000px)
- Portrait orientation
- Professional appearance
- Will be displayed with artistic paint overlay effects

### 2. Update Resume
Add your latest resume at:
```
src/assets/resume/Ifedayo_Ademodi_Resume.pdf
```

### 3. Customize Content
Edit content in `src/app/landing/landing.component.ts`:
- Update skills arrays
- Modify experience details
- Add/remove projects
- Update social links

### 4. Adjust Styling
Modify styles in `src/app/landing/landing.component.scss`:
- Change color schemes
- Adjust spacing and layout
- Customize animations
- Update responsive breakpoints
